leetCode
====

[leetcode](https://leetcode.com/problems/)

leetcode est un site qui répertorie plus de **3 000** problèmes d'algorithmique
l'objectif de ce quitte est de proposer une solution à tous ces problèmes.

il y a 3 types de défi <span style="color:green">
Easy</span> <span style="color:orange">Hard</span> <span style="color:red">Hard</span>


Pour commencer, je vais suivre une [roadmap](https://neetcode.io/roadmap) qui propose un ordre de résolution pour 150 défis.

certains défis ont besoin de la version payante pour les réaliser.
